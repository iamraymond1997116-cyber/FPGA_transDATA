# V6.3 项目术语速查

## 极性反直觉

| 信号 | 实际语义 | 易错 |
|:---|:---|:---|
| `sensor_power = 1'b1` | 传感器 **OFF**（断电） | PMOS 反相，1=OFF |
| `sensor_power = 1'b0` | 传感器 **ON**（上电） | 0=ON |
| `LED1~LED4` 亮 = 0 | **低电平点亮** | 以为高电平亮 |
| `KEY` 按下 = 0 | 电平拉低 | 以为按下 = 1 |

## 版本号

只在 `rtl/transient_puf_v60_top.v` 定义一次。子模块通过参数接收。
