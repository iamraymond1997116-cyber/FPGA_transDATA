# 变换层扩展 — TODO

按推荐优先级排列：

- [ ] **相位谱**（丢弃幅度，保留相位）— 幅度谱受信号能量影响大，而相位谱对微小的时序差异极度敏感——传感器物理差异可能更多体现在相位上。零额外计算，FFT 结果直接复用。

- [ ] **互谱 CH1×CH2 / 传递函数 CH2÷CH1** — 双通道的比值可以抵消共模噪声，突出两通道之间的差异。这可能对 B2-6 vs B2-8 这种极相似对有奇效。

- [ ] **Hilbert 变换 → 瞬时包络 + 瞬时频率** — 把信号分解成幅度包络和瞬时频率两条轨迹。传感器物理差异可能更多体现在包络衰减速率和频率调制模式上，而不是幅度谱的具体形状。

- [ ] **小波变换** — 针对瞬态信号最自然的工具。FFT 是固定分辨率，小波在低频高频率分辨率、高频高时间分辨率，特别适合捕捉 MODE=3 那种快速翻转中的瞬态特征。

- [ ] **自相关函数** — 直接反映信号内部的周期性和时间常数。不同传感器的阻尼系数、谐振频率在自相关上的衰减速度不同。

- [ ] **倒谱（Cepstrum）** — "频谱的频谱"，对信号中的回声/反射模式敏感。如果传感器内部有机械谐振或声学反射腔，倒谱能捕捉到。

- [ ] **时频图谱（STFT 2D）+ 卷积特征** — 把 128 点 STFT 变成 2D 时频图，用 CNN 提特征。但这需要更多数据。

- [ ] **双谱（Bispectrum）** — 对传感器非线性行为敏感，检测频率分量之间的相位耦合。
