# 专利大纲：双通道滑动窗口稳定性检测器

## 拟定名称

一种面向传感器瞬态身份认证的双通道滑动窗口稳定性检测装置及方法

## 建议定位

**独立申请或专利01的强从属权利要求**。这是硬件采集链中的关键创新模块，为后续特征提取提供质量门控。三件已有专利均未涉及时域稳定检测的硬件实现。

## 要解决的技术问题

传感器上下电瞬态响应的采集窗口需要精确界定——过早截断会丢失有效信息，过晚采集会引入稳态噪声。传统方法使用固定延时或外部触发，难以适应不同传感器的瞬态持续时间差异。同时，双通道采集时两个通道的稳定时刻可能不一致，简单的单通道检测会导致通道间数据错位。

本方案要解决的问题是：如何在采集过程中实时检测传感器输出何时进入稳定状态，并确保双通道的稳定判定严格同步。

## 核心发明点

1. **双通道独立滑动窗口检测**：对CH1和CH2分别维护独立的滑动窗口，每个窗口检查连续N个采样点（N=16）是否全部落在±T LSB（T=20）的容差带内。
2. **动态窗口参考值重置**：当当前采样点超出容差带时，窗口参考值立即重置为当前采样值，计数器归零，开始新的稳定窗口计数。这允许检测器适应非单调的瞬态波形。
3. **双通道联合稳定判定（AND逻辑）**：仅当CH1和CH2同时满足稳定条件时，才记录稳定点索引。任一通道未稳定则整体不标记，确保双通道数据的时序对齐。
4. **稳定点超出窗口的异常标记**：如果256点采集完成时仍未检测到双通道联合稳定，标记`stable_point=256`（超出缓冲区），作为采集质量异常的指示。

## 系统组成

- **双通道采样接口**：用于接收CH1和CH2的ADC采样数据；
- **CH1滑动窗口检测器**：用于检查CH1采样序列的稳定性；
- **CH2滑动窗口检测器**：用于检查CH2采样序列的稳定性；
- **联合判定器**：用于在CH1和CH2同时满足稳定条件时输出稳定点索引；
- **异常标记器**：用于在采集完成但未检测到联合稳定时输出异常指示。

## 独立权利要求骨架

### 装置权利要求

一种面向传感器瞬态身份认证的双通道滑动窗口稳定性检测装置，包括：

1. 第一通道稳定性检测模块，用于对第一通道的采样序列进行滑动窗口稳定性检测，当连续预定数量的采样点均落在第一窗口参考值的预定容差范围内时，判定第一通道稳定；
2. 第二通道稳定性检测模块，用于对第二通道的采样序列进行滑动窗口稳定性检测，当连续预定数量的采样点均落在第二窗口参考值的预定容差范围内时，判定第二通道稳定；
3. 联合判定模块，用于仅在所述第一通道和所述第二通道均被判定为稳定时，输出稳定点索引；
4. 动态重置模块，用于在任一采样点超出对应窗口的预定容差范围时，将该通道的窗口参考值重置为当前采样值，并将稳定计数归零。

### 方法权利要求

一种传感器瞬态响应采集中的双通道稳定性检测方法，包括：

1. 对第一通道和第二通道的采样数据进行同步采集；
2. 对所述第一通道的采样数据进行滑动窗口稳定性检测；
3. 对所述第二通道的采样数据进行滑动窗口稳定性检测；
4. 当所述第一通道和所述第二通道同时满足稳定性条件时，记录当前采样点索引作为稳定点；
5. 当任一采样点超出对应通道的当前容差带时，重置该通道的窗口参考值并重新开始稳定计数。

## 从属权利要求方向

- 所述预定数量为16，所述预定容差范围为±20 LSB。
- 所述稳定性条件为：连续16个采样点的值与对应窗口参考值之差均不超过20 LSB。
- 所述装置还包括采集完成检测器，用于在采集点数达到256时结束采集；若此时未输出稳定点索引，则输出异常指示。
- 所述第一通道和第二通道为同一传感器的双路差分输出或两个独立传感器的输出。
- 所述稳定点索引用于界定传感器瞬态响应的有效采集窗口。

## 风险与规避

- 滑动窗口稳定性检测在信号处理领域有现有技术，需要强调本发明的特定应用场景（"传感器瞬态身份认证"）和特定硬件实现（"双通道联合AND判定"+"动态参考值重置"）。
- 避免将通用滑动窗口检测作为核心创新，要强调双通道同步检测对身份采集质量的保障作用。
- 容差参数（16样本、±20 LSB）可根据不同传感器类型调整，权利要求中应使用上位表述。
